# RENPY_Skel

A reusable Ren'Py project skeleton template. Fork this repo for each new visual novel project.

## What's Included

- **Organized folder structure** for images, audio, fonts, and scripts
- **Template `.rpy` files**: `script.rpy`, `definitions.rpy`, `effects.rpy`
- **EasyRenPyGui** (by Feniks) - clean, modular GUI screens and styles
- **Achievements System** (by Feniks) - popups, gallery, progress tracking, Steam support
- **Dialogue Bleeps Pack** (by dmochas) - 30 typing sound effects for character voices

## Quick Start

1. Fork this repo
2. Edit `game/options.rpy` (set game name, build name, save directory)
3. Edit `game/definitions.rpy` (add your characters)
4. Edit `game/script.rpy` (write your story)
5. Drop assets into `game/images/` and `game/audio/`

See **[USER_GUIDE.md](USER_GUIDE.md)** for full documentation.
